Config 2024: Improve your design system with confidence through A/B testing (Steve Dodier-Lazaro)

In this session, Steve will discuss the why and the how of A/B testing Design System components in a Web environment. Steve will present the A/B testing setup at Back Market and how it was built to blend into existing workflows for Design System contributions and Web app A/B testing. They'll discuss the technical challenges to A/B testing Design Systems and explain how to overcome them. They'll also touch on how A/B tests can help align Design System team roadmaps with product leadership goals.

Featuring: Steve Dodier-Lazaro, Design System Engineer, France Télévision
