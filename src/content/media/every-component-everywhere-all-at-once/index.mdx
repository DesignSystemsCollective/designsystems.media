There is an explosion of components across GitHub as we build more and more of our UIs in React. Over at Primer, we wondered how difficult it would be to see how these components are being used. In this talk, Josh Black, Senior Software Engineer at Github, will show you how to use a tool called Primer Query to search across the GitHub universe to answer life’s most important component usage questions.
